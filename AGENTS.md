# AGENTS.md

## 项目目标

这是一个原创 IP 的网页端怪物收集肉鸽游戏。当前阶段只做本地单机最小可玩 Demo，不加入登录、充值、广告、服务器或联机。

## 技术规则

- 优先使用 React + TypeScript + Vite。
- 前期无后端，所有存档使用 `localStorage`。
- 怪物、技能、遗物、关卡等内容尽量放在 `src/data` 配置中。
- 游戏核心类型与逻辑放在 `src/game`，界面组件放在 `src/components`。
- 不把关键数值硬编码在组件里，组件只负责展示和触发动作。
- 保持小步提交式开发：每次只扩展一个明确系统。

## 内容规则

- 所有怪物、技能、遗物、世界观、UI 文案必须原创。
- 禁止使用任何宝可梦名称、素材、设定、UI、音乐、招式名或类似角色设计。
- 不模仿现有知名怪物的轮廓、配色、进化链、球形捕捉器、属性克制表或图鉴表达。
- 捕获、收集、回合制、爬塔、随机奖励属于通用玩法，可以使用，但表达方式必须原创。

## 代码风格

- TypeScript 开启严格模式。
- 命名清晰，优先组合小函数。
- UI 文案用中文。
- 新系统优先写成数据驱动，方便后续扩展。
- 避免过早抽象，Phase 1 以可读、可玩、可扩展为主。
