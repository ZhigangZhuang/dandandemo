# CONTENT_RULES.md

## 原创 IP 规则

本项目必须建立独立世界观与美术表达。怪物可以是奇异生命、塔内生态、晶体生物、菌丝构造、机械残响等原创方向，但不能靠近任何现有知名怪物收集 IP 的具体表达。

## 禁止内容

- 禁止使用宝可梦或其他知名怪物 IP 的名称、素材、音乐、音效、图标、招式名、道具名、角色名、地名。
- 禁止使用相似到会被识别为改名版本的角色设计。
- 禁止使用类似“精灵球”的捕获器外观或命名。
- 禁止复刻宝可梦式属性表、进化链、图鉴 UI、战斗 UI、城镇道馆结构。
- 禁止将现有角色进行换色、改角、换名字后使用。
- 禁止直接照搬“性格修正表”“属性克制表”“进化方式”等具体表达；可做通用成长系统，但必须重新命名、重新设定、重新平衡。

## 可使用的通用玩法

以下是通用游戏机制，可以使用，但需要原创表达：

- 收集怪物。
- 回合制战斗。
- 捕获敌人。
- 随机奖励。
- 肉鸽爬塔。
- 本地存档。
- 图鉴或收藏页面。

## 命名规则

- 怪物名使用原创中文名，不使用知名 IP 的音译、谐音或结构。
- 技能名避免“十万伏特”“火花”“水枪”等高度识别的既有招式。
- 遗物名应体现本项目世界观，例如塔、壳、回响、尘晶、脉纹、潮痕等。
- 成长系统使用“气质”“蜕变”“灵种”“生命纹路”等原创表达，不使用受保护 IP 的专有 UI 和术语组合。
- 捕捉道具使用“灵契球”等原创命名和外观，不使用红白球、中心按钮、投球捕捉动画或相似视觉符号。

## 美术方向

- Phase 1 使用原创像素风宠物轮廓，不使用外部素材。
- 怪物可以基于真实动物加属性改造，例如龟 + 水晶灯、小鹿 + 草叶岩角、沙狐 + 水铃尾。
- 避免红白球、经典御三家动物轮廓、黄色电气鼠类、乌龟炮管、喷火龙等可联想到具体角色的设计元素。

## 审查清单

新增任何怪物、技能、遗物前，检查：

- 名称是否原创。
- 外观是否不像现有知名角色。
- 描述是否没有借用受保护世界观。
- UI 是否没有复刻现有游戏。
- 玩法表达是否服务于本项目自己的主题。
